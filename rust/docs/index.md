# Rust Module Documentation

This directory documents Rust-specific modules, patterns, and conventions for
LCS.

## Scope

- Module responsibilities and boundaries
- Public APIs and internal contracts
- Error handling patterns
- Testing expectations
- Build and lint rules

## Module Documentation Template

Use the following headings for each module document:

- Purpose
- Responsibilities
- Public API
- Dependencies
- Errors
- Tests
- Notes

## Location

Create one file per module under `rust/docs/`.
