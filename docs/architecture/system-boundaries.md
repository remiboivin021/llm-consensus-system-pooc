# System Boundaries

## In Scope

- Consensus engine and scoring logic
- API layer and request handling
- LLM execution adapters (Python)
- Observability (metrics and logging)

## Out of Scope

- Model training or fine-tuning
- Client UI applications
- External platform billing or user management

## External Dependencies

- LLM providers or runtimes
- Optional storage for run logging
- Monitoring backends for metrics
