# Security Overview

This section describes the security posture of LCS.

## Goals

- Protect inputs and outputs
- Reduce attack surface
- Support responsible disclosure

## Policies

- See `SECURITY.md` for reporting and response.
- Avoid logging sensitive data by default.
- Use encryption in transit.
