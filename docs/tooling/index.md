# Tooling Summary

LCS uses Rust and Python tooling. This section lists the standard toolchain.

## Rust

- rustup (toolchain management)
- cargo (build and test)
- rustfmt (formatting)
- clippy (linting)

## Python

- Python 3.11
- uv (package management)
- ruff (linting and formatting)
- pytest (testing)
